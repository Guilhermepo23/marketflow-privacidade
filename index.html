Política de Privacidade – MarketFlow

Última atualização: 21/08/2025

1. Informações que coletamos

  • Dados fornecidos pelo usuário: nome, e-mail, telefone.
  • Dados de uso: registros de vendas, itens de estoque, relatórios.
  • Dados técnicos: modelo do dispositivo, sistema operacional, versão do app.

2. Como utilizamos as informações

  • Operar e manter o funcionamento do aplicativo.
  • Personalizar a experiência e melhorar recursos.
  • Gerar relatórios e estatísticas internas.
  • Enviar notificações e comunicações importantes (quando autorizado).

3. Compartilhamento de dados

O MarketFlow não vende, aluga ou compartilha dados pessoais com terceiros, exceto quando exigido por lei.

4. Armazenamento e segurança

Os dados são armazenados de forma segura e protegidos contra acessos não autorizados.

5. Direitos do usuário

  • Solicitar acesso, correção ou exclusão de seus dados pessoais.
  • Revogar consentimentos concedidos.
  • Solicitar portabilidade dos dados (quando aplicável).
